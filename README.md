Overview:
This project is a deep learning-based system designed to detect plant diseases from leaf images using a Convolutional Neural Network (CNN). It provides an automated and efficient way to classify plant health conditions, helping in early diagnosis and reducing dependency on manual inspection. The system integrates image preprocessing, model prediction, and a simple user interface for practical usage.

Problem:
Farmers often face difficulty in identifying plant diseases at an early stage due to lack of expertise and limited access to proper diagnostic tools. Delayed or incorrect identification can lead to significant crop loss and reduced productivity.

Solution:
The system uses a CNN model trained on labeled plant leaf images to classify diseases. Images are preprocessed and augmented using techniques such as resizing and normalization to improve model performance. The trained model predicts the disease class, and a user-friendly interface allows users to upload images and get instant results, enabling timely action.

Features:
1. CNN-based image classification
2. Image preprocessing and data augmentation
3. Real-time prediction through user interface
4. Multi-class disease detection
5. Visualization of results

Technologies Used:
Software:
1. Python
2. TensorFlow / Keras
3. NumPy
4. Pandas
5. Matplotlib
6. Streamlit

System Workflow:
1. User uploads plant leaf image
2. Image is resized and preprocessed
3. CNN model processes the image
4. Model predicts disease class
5. Output is displayed to the user
